¶前言

  2. 熟悉 linux 开发环境,掌握 tomcat、mysql 服务器集群的安装与配置。

  4. 掌握数据库主从复制,读写分离技术。
  5. 理解缓存原理,掌握缓存在程序开发的原理。
  6. 掌握 web 站点的动静分离技术。
