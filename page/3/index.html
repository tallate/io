  1. 掌握高并发站点的系统架构方法。
  2. 熟悉 linux 开发环境,掌握 tomcat、mysql 服务器集群的安装与配置。
  3. 理解 web 服务器的负载均衡原理,掌握配置方法。
  4. 掌握数据库主从复制,读写分离技术。

  6. 掌握 web 站点的动静分离技术。
