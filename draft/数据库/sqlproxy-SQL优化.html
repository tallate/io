¶MySQL优化

  1. 腾讯面试：一条SQL语句执行得很慢的原因有哪些？—不看后悔系列
    偶尔很慢的原因：MySQL刷新脏页（内存->redolog->磁盘）、锁抢占（表锁、行锁，使用show processlist排查）
    长期很慢的原因：未命中索引、选错索引（采样->基数大走索引基数小走全表扫描）。
    索引分析：
    select * from t force index(a) where c < 100 and c < 100000;
    show index from t;
    analyze table t;
